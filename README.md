Best Shop is a simple e-commerce web application specializing in suitcases and luggage products. It is developed using HTML, SCSS for styling, and plain JavaScript for interactivity.

---

## Prerequisites

Make sure you have installed:

- **Node.js** (v16+ recommended)  
- **npm** (comes with Node)

---

## Installation

Clone the repository and install dependencies:

git clone https://gitlab.com/georgiana17stanciu-group/georgiana17stanciu-project.git
cd fundamentals-project-template

Please make sure that sass and live-server are installed. Here are the commands to install them:
    - npm install --save-dev live-server
    - npm install --save-dev sass

After installing dependencies we can install the project with command: npm install

---

## Run

Run command 'npm run dev'
